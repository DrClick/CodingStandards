Things to take note of are the [HttpGet] and [HttpPost] attributes on the methods and that the post method expects 1 parameter of type HelloWorldModel. That method fakes a long call so we can see the front end work and returns a JSONResponse object. Here are a look at those two POCCO objects,

Now we just have to handle the submitting of a basic HTML form. Armed with require, it is very simple. NOTE: We should be creating a view here, but as it is 10 minutes before I am supposed to show this, it is what it is!

(function () {
    var Labelmaster = null;

    //Take notice of this sytax
    require(['labelmaster/labelmaster'], function (lm) {
        Labelmaster = lm;
        _init.call(this);
    });

    function _init() {
        $("#mainForm").on("submit", function (evt) {
            evt.preventDefault();

            //this is the magic!
            Labelmaster.postForm(evt.target, "/home/helloWorld");
        });
    }//end init

    amplify.subscribe("HelloFromTheServer", _listenForServerResponse);
    function _listenForServerResponse(data){
        $("#response").html(data);
        $("#response").addClass("alert alert-warning");
    }
})();

The key things to note here are we are requiring Labelmaster, capturing the form submit and using the labelmaster method. Finally we are listening for response and taking action when the server comes back. This lets us be very responsive to the user and fail gracefully. More to come!
