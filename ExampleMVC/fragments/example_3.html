From a Razor view, add the following to the @scripts section

	<script type="text/javascript">
        (function () {
            var View = null;
            
            //Take notice of this sytax
            require(['app/home/UsingRequire'], function (view) {
                View = view;
                _init.call(this);
            });

            function _init() {
                var view = new View(
                    {
                        clickable       : ".UsingRequire .clickable",
                        templateDisplay : ".UsingRequire .templateDisplay"
                    },
                    {
                        clickColor: ["#002244", "#69BE28", "#A5ACAF"]
                    }
                );
            }//end init
        })();

Things you should notice:

  1. Self closing function. This keeps you from accidentally polluting the global namespace.
  2. The use of Require to load your script asynchronously.
  3. The first argument of a view will always be the DOM handles. This separates the DOM from the javascript module and address a major source of brittleness
  4. The second argument isan options argument. Which can be used to override DEFAULT OPTIONS defined on the class itself.
