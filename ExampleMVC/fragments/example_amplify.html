Amplify is a amoung other things a pub/sub library for javascript. Check out the documentation here

We are going to use it as our eventing engine. When should you use it? When ever you make a call to the server. When else? Often!

Here is a demo

	(function(){
		$(".person.leader").click(function(){
			alert("I am going to publish an event named leader.clicked");
			amplify.publish("leader.clicked", {creedo:"So Say We All"});
		});

		//here we are going to listen for the leader.clicked event
		amplify.subscribe("leader.clicked", function(data){
			$(".person.minion").html(data.creedo);
		});

		//the power really comes when you realize you can have multiple listeners
		//this is important when when you have multiple sections of the screen
		//that must respond based on event.
		amplify.subscribe("leader.clicked", function(data){
			$(".person.minion").css("background-color", "gainsboro");
		});
	})();

As you can see, it's easy to use and intuitive. The real power of course is async events like a post or get. Break away from inlined continuation and start thinking pub/sub!
